# ADR-003: Supabase over Self-Hosted PostgreSQL

## Decision
**Will:** Use Supabase managed PostgreSQL
**Will NOT:** Self-host on AWS RDS or DigitalOcean

## ClaudeCode&OnlyMe: 4/4 YES
- Real TODAY: YES
- Simplest: YES (managed, <5min setup)
- 2-person: YES (zero ops)
- Value NOW: YES (removes infrastructure burden)
